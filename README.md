# 📚 한국사 퀴즈 앱

> **완자 한국사 + 해냄에듀 한국사1·2 교과서 기반** 재미있는 한국사 학습 퀴즈 게임

---

## 🎯 프로젝트 목표

2026년 8월 11일 한국사 시험을 앞두고, 방대한 한국사 내용을 게임 형식으로 재미있게 학습하기 위해 제작되었습니다.

---

## ✅ 구현된 기능

### 🎮 게임 시스템
- **라운드 시스템**: 1라운드 = 20문제 (전체 100문제 중 랜덤 출제)
- **틀린 문제 재출제**: 틀린 문제는 해당 라운드 마지막에 다시 출제
- **맞출 때까지 도전**: 맞출 때까지 반복 출제 (포기 불가)
- **오답 보관함**: 틀린 문제를 누적 저장하여 오답 복습 모드로 재도전 가능

### 💰 포인트 시스템
| 조건 | 포인트 |
|------|--------|
| 처음 맞추기 | +100점 |
| 틀린 후 재도전 성공 | +50점 |
| 오답 | -10점 |
| 라운드 클리어 보너스 | +200점 |
| 퍼펙트 클리어 (틀린 문제 0) | +700점 |

### 📖 학습 기능
- **정답 해설**: 맞추든 틀리고 맞추든 해설 확인 가능
- **난이도 표시**: 쉬움/보통/어려움으로 구분
- **단원/주제 표시**: 어느 단원의 문제인지 확인 가능
- **D-DAY 카운트다운**: 시험까지 남은 날짜 실시간 표시

### 💾 데이터 저장
- LocalStorage를 이용한 게임 진행 상황 저장 (브라우저 닫아도 유지)
- 누적 포인트, 정답률, 틀린 문제 보관함 저장
